== Description ==

##**ENGLISH**

>WPFacebookCurtir is a simple way to easily add automatically a Facebook like button on your wordpress posts.

>Just install the plugin, activate it, and enjoy ;) WPFacebookCurtir will add a Facebook like on top of each wp post, just below the title and date of wp post.

>To see this wordpress plugin facebook like working and extras info, please visit http://blog.idealmind.com.br/wordpress/insereiframe-a-simple-wordpress-plugin-to-insert-iframe-in-posts/

##**PORTUGUÊS**

>WPFacebookCurtir insere automaticamente um botão curtir do Facebook em cada um dos posts do seu blog wordpress.

>Apenas instale o plugin, ative-o e pronto! WPFacebookCurtir adicionará um botão curtir do Facebook no topo de cada post, logo abaixo do título e da data de postagem.

>Para ver informações extras e o wordpress plugin facebook curtir em uso, visite http://blog.idealmind.com.br/wordpress/insereiframe-a-simple-wordpress-plugin-to-insert-iframe-in-posts/

== Installation ==
## **ENGLISH**

>To install WPFacebookCurtir in your Wordpress blog, follow the steps below:

1. Install it using Wordpress Panel Control or Download and unzip WPFacebookCurtir.zip and follow the steps below:
2. Upload `WPFacebookCurtir.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. See any of your posts and enjoy ;)

## **PORTUGUÊS**

>Siga os passos abaixo para instalar o WPFacebookCurtir no seu blog Wordpress:

1. Instale pelo Painel de Controle do Wordpress ou faça o download e descompacte WPFacebookCurtir.zip e siga os passos abaixo:
2. Faça o upload do arquivo `WPFacebookCurtir.php` para a pasta `/wp-content/plugins/`
3. Ative o plugin através do menu 'Plugins' do WordPress
4. Veja qualquer um de seus posts e pronto!
